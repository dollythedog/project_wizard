## Instructions

Generate a complete project charter that:

1. **Expands the problem statement** into clear problem/opportunity sections
2. **Develops comprehensive scope** with explicit in-scope and out-of-scope items
3. **Creates measurable success criteria** based on the benefits provided
4. **Identifies realistic risks** specific to this project with mitigation strategies
5. **Defines deliverables** that align with the solution and requirements
6. **Establishes timeline** consistent with the provided dates
7. **Details collaboration needs** based on resource requirements

Use the provided information as the foundation, expanding brief inputs into well-structured charter sections. Maintain professional PM terminology and ensure all metrics are specific and measurable.

Output in the exact template format with all sections completed.
